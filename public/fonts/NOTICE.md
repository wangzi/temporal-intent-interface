# Bundled webfont attribution

## iA Writer Duo S

- Files: `iAWriterDuoS-{Regular,Italic,Bold,BoldItalic}.woff2`
- Source: https://github.com/iaolo/iA-Fonts (`iA Writer Duo/Webfonts`)
- © iA Inc. Released under the SIL Open Font License.
- Used here as the cross-platform monospace fallback (non-Apple devices)
  behind the system SF Mono. Per iA's request: "If you fork or use our
  fonts, please reference iA Writer clearly."

SF Mono itself is **not** bundled — it is referenced as a system font and
resolves only on Apple platforms where it is installed.
